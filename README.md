# Proyecto Integrador  
## El impacto de las nuevas tecnologías en la sociedad: visualización del futuro

### Información general del proyecto
Este proyecto integrador consiste en el desarrollo de una aplicación web educativa e interactiva orientada al análisis del impacto de las nuevas tecnologías en la sociedad contemporánea y su proyección hacia el futuro.

El proyecto integra contenidos de las cuatro unidades de la asignatura y fue desarrollado como un software funcional de carácter académico, con el propósito de visualizar de manera clara, organizada e interactiva cómo tecnologías emergentes como la inteligencia artificial, la robótica, la realidad virtual y aumentada, y la automatización están transformando la vida social, económica, educativa y laboral.

---

## 1. Nombre del proyecto
**El impacto de las nuevas tecnologías en la sociedad: visualización del futuro**

---

## 2. Descripción general del problema
En la actualidad, la adopción acelerada de nuevas tecnologías ha generado cambios profundos en la forma en que las personas trabajan, estudian, se comunican y toman decisiones. Sin embargo, muchas veces estos cambios ocurren más rápido que la comprensión social de sus efectos.

Este proyecto busca atender la necesidad de analizar de manera amplia y reflexiva el uso de las nuevas tecnologías y su impacto en la sociedad, permitiendo identificar tanto sus beneficios como sus riesgos, implicaciones y limitaciones.

---

## 3. Propósito del proyecto
Visualizar el impacto de las nuevas tecnologías en la sociedad y reflexionar sobre su potencial futuro mediante el desarrollo de una aplicación web interactiva que permita presentar información académica de forma clara, dinámica y organizada.

---

## 4. Objetivo general
Desarrollar un software funcional en formato web que permita exponer, comparar y analizar el impacto social de las nuevas tecnologías, integrando los conocimientos adquiridos en las cuatro unidades de la materia.

---

## 5. Objetivos específicos
- Analizar el impacto de la inteligencia artificial, la robótica, la realidad virtual y aumentada, y la automatización.
- Identificar beneficios, riesgos, impacto social y proyecciones futuras de cada tecnología.
- Comparar tecnologías para reconocer relaciones, diferencias y posibles escenarios de convergencia.
- Presentar conclusiones fundamentadas sobre el papel de estas tecnologías en la sociedad.
- Exponer las implicaciones y limitaciones del análisis realizado.
- Integrar conceptos teóricos y prácticos de programación en un producto digital funcional.

---

## 6. Tipo de software desarrollado
Se desarrolló una **aplicación web educativa e interactiva**, diseñada para ser visual, funcional y accesible desde navegador.

El sistema fue construido como una herramienta de consulta y análisis académico, orientada a presentar contenidos de forma estructurada y con elementos de interacción para facilitar la comprensión del tema.

---

## 7. Funcionalidades principales del software
La aplicación web incluye las siguientes funcionalidades:

- Página principal con presentación del tema
- Descripción del problema abordado
- Propósito del proyecto
- Sección de análisis de tecnologías emergentes
- Apartados específicos para:
  - Inteligencia Artificial
  - Robótica
  - Realidad Virtual y Aumentada
  - Automatización
- Desarrollo de cada tecnología con:
  - descripción
  - beneficios
  - riesgos
  - impacto social
  - proyecciones futuras
- Matriz comparativa de impacto entre tecnologías
- Sección de conclusiones finales
- Sección de implicaciones sociopolíticas
- Sección de limitaciones del estudio
- Navegación interactiva y diseño responsive para diferentes dispositivos

---

## 8. Tecnologías utilizadas
Para el desarrollo del proyecto se emplearon tecnologías web básicas y accesibles:

- **HTML5** para la estructura de la página
- **CSS3** para el diseño visual, presentación y responsividad
- **JavaScript** para la interacción y la comparación dinámica de tecnologías
