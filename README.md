# El impacto de las nuevas tecnologías en la sociedad: visualización del futuro

Proyecto universitario enfocado en la investigación, análisis y exposición académica sobre cómo las tecnologías emergentes están redefiniendo el futuro de la humanidad. Esta plataforma sirve como un recurso interactivo de una sola página (Single Page Application - estática) para explorar dilemas éticos, impactos sociales y proyecciones futuras.

## Características Principales

- **Diseño Moderno y Académico:** Una interfaz pulida y seria que invita a la lectura profunda.
- **Responsive Design:** Completamente adaptable a dispositivos móviles, tablets y escritorios.
- **Navegación Interactiva:** Menú fijo (sticky) y menú hamburguesa para dispositivos móviles.
- **Componente de Pestañas (Tabs):** Para explorar limpiamente las 4 categorías tecnológicas principales sin recargar la página.
- **Herramienta de Comparación:** Permite al usuario seleccionar dos tecnologías para descubrir sus sinergias e impactos combinados mediante lógica en JavaScript.
- **Código Limpio y Comentado:** Ideal para estudiantes y desarrollo colaborativo.

## Estructura del Proyecto

El proyecto está construido sin frameworks pesados en el frontend para garantizar máxima velocidad y simplicidad.

- `index.html`: Estructura semántica, accesibilidad y contenido de la exposición.
- `style.css`: Hojas de estilo personalizadas con variables CSS, animaciones y Media Queries.
- `script.js`: Lógica de interacción para la navegación, pestañas (tabs) y la matriz comparativa de tecnologías.
- `README.md`: Documentación del proyecto.

## Tecnologías Utilizadas

- **HTML5**
- **CSS3** (Variables nativas, Flexbox, CSS Grid)
- **JavaScript** (Vanilla JS, ES6)

## Instrucciones para Ejecutar Localmente

1. Clona el repositorio o descarga los archivos.
2. Abre la carpeta del proyecto.
3. Puedes abrir el archivo `index.html` directamente en cualquier navegador web moderno (Chrome, Firefox, Safari, Edge).
4. Alternativamente, si tienes un servidor local como Live Server (extensión de VSCode) o estás usando el entorno proporcionado, simplemente inicia el servidor apuntando al archivo `index.html`.
